# templates

## Installation

Télécharge le .zip 

`npm install` - Installer les dépendances dans le dossier : `./compiler`
`npm run compiler` - Lancer le programme (déjà configurer dans le package.json)

`npm init` - Initialisation du projet dans le dossier parent
